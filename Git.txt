NAME 

   giteveryday - A useful minimum set of commands for Everyday Git


SYNOPSIS

     Everyday Git With 20 Commands Or So

DESCRIPTION
      
           Git users can broadly be grouped into four categories for the purposes of describing here a small of 
useful command for everyday Git

* Individual Developer (Standalone) commands are essential for anybody who makes a commit,even for somebody who wroks alone.

*If you work with other people,you will need commands listed in the Individual  Developer(Participant) section as well.

* People who play the Integrator role neede to learn some more commands in additon to the above .

*Repository  Administration commands are for system administrators who are responsible for the care and feeding of Git repositories.

Individual Developer(Standalone)

A standalone individual developer does not exchange patches with other people,and works alone in a single repository,using the following commands.

* git -init[1] to create a new repository.

     NAME 
             git-init -Create an empty Git repository or reinitialize an existing one 

     SYNOPSIS
             git init [-q  | --quiet] [--bare] [--template=<template-directory>]
[--separate-git-dir <git-dir>] [--object-format=<format>]
